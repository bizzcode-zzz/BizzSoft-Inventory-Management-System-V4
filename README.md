## 📖 About

BizzSoft Inventory Management System V4 is a modern inventory management application built with **Laravel 12**.

The system allows businesses to manage products, categories, suppliers, purchases, sales, reports, and inventory using a clean Software Engineering architecture.

This project was developed as a real-world learning project focused on building production-quality software rather than only learning CRUD operations.

# 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

```bash
cd YOUR_REPOSITORY
```

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

Configure your database inside **.env**

Run migration

```bash
php artisan migrate
```

Run seeders (optional)

```bash
php artisan db:seed
```

Start Laravel

```bash
php artisan serve
```
